# 入门指南

## 先备知识

Box Pleating Studio 并非为折纸初学者设计。
为了充分理解它所产生的展开图并能够折叠成品，
你必须对单轴箱形褶基本形的理论有相当程度的了解。
你可以在 Robert J. Lang 出色著作的第 13 和 14 章中找到所需的一切知识
（见参考文献 [[1]](/zh-CN/notes#参考文献)），或参考[资源](/zh-CN/resources)页面中列出的其他教程。

## 如何获取应用

只需访问 [https://bpstudio.abstreamace.com/](https://bpstudio.abstreamace.com/)
即可启动 BP Studio 应用，无需下载！
为了获得最佳性能，强烈建议使用 Chrome 浏览器来访问网站。
无论如何，请将你的浏览器更新到最新版本，
因为 BP Studio 使用了许多新的网页功能，无法在旧版浏览器上运行。

如果你使用 Chrome（iPhone 以外的设备）、Edge（桌面版）或 Safari（iPhone）访问网站，
它也会提示你可以将 BP Studio 安装为独立应用，并离线使用。
只需按照指示安装即可。
这将提供最佳的用户体验，尤其是在移动设备上。
