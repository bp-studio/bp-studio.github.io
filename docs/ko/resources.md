# 리소스

다양한 기여자들이 만든 리소스 목록으로, 제가 알고 있는 것들을 정리했습니다.

## 강연

- [Introduction to Box Pleating Studio](https://youtu.be/CMgli6Ul7CE) — 저자 본인이 2022년 행사 "Origami Matemáticas II"에서 Sociedad Boliviana de Origami를 위해 진행한 강연 (Jaciara Grzybowski의 스페인어 번역).

## 튜토리얼 영상

- [Origami Designing in BPStudio - Scorpion](https://youtu.be/PtJE0rankHc) — [cExplr](https://www.youtube.com/channel/UC-HBfABJOMSUhkgdaNh36lA)이 제작.
- [Bp Studio and Treemaker Crash Course \| Origami design softwares](https://youtu.be/OCDR8cRvKAY) — [The Plant Psychologist](https://www.youtube.com/channel/UCJPCcyo8mTKdIU8oLMv5WpQ)이 제작.
- [Live designing with BP Studio](https://youtu.be/mOXP2FhGHi8) — [Origambel](https://www.youtube.com/channel/UCgRy3dFG5qVFmu0QHZ5uIsQ)이 제작.
- [DOAZ#3 폴리곤 패킹과 박스 플릿의 이해 (T2) [자막]](https://youtu.be/4SvFRTyEyqM) — [Paper Shaper Lee U](https://www.youtube.com/@papershaperleeu5586)이 제작 (한국어, 영어 자막 포함). BP Studio의 기초와 피타고라스 스트레치를 간략하게 소개.
- [기미로는 종이접기 할 때 어떤 프로그램을 쓸까?](https://youtu.be/Za7KEeic24g) — [기미로](https://www.youtube.com/@origamikimiro) (Origami_Kimiro)이 제작 (한국어, 영어 자막 포함). 여러 주요 종이접기 소프트웨어 중 BP Studio를 간략하게 소개. 기미로는 BP Studio 활용의 최고 전문가 중 한 명.
