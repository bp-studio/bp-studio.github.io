# 시작하기

## 사전 지식

Box Pleating Studio는 종이접기 초보자를 위한 앱이 아닙니다.
이 앱이 생성하는 패턴을 완전히 이해하고 실제로 접을 수 있으려면,
단축 박스 플리팅 기본형의 이론에 상당히 익숙해야 합니다.
Robert J. Lang의 훌륭한 책의 13장과 14장에서 필요한 모든 내용을 찾을 수 있습니다
(참고문헌의 [[1]](/ko/notes#참고문헌)을 참조하세요). 또는 [리소스](/ko/resources) 페이지에 나열된 다른 튜토리얼을 확인해 보세요.

## 앱 이용 방법

[https://bpstudio.abstreamace.com/](https://bpstudio.abstreamace.com/)을 방문하기만 하면
BP Studio 앱을 바로 실행할 수 있으며, 별도의 다운로드가 필요하지 않습니다!
최상의 성능을 위해 Chrome 브라우저 사용을 강력히 권장합니다.
BP Studio는 많은 새로운 웹 기능을 활용하기 때문에 구형 브라우저에서는 작동하지 않으므로,
브라우저를 항상 최신 버전으로 업데이트해 주세요.

Chrome(iPhone 이외의 기기), Edge(데스크탑), 또는 Safari(iPhone)로 웹사이트를 방문하면
BP Studio를 독립 실행형 앱으로 기기에 설치하고 오프라인으로 사용할 수 있다는 안내가 표시됩니다.
안내에 따라 설치하면 됩니다.
특히 모바일 기기에서 최상의 사용자 경험을 제공합니다.
