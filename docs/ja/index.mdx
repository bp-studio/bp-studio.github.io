## はじめに

ボックスプリーツは、現代の超複雑な折り紙デザインの標準的なフレームワークと言えます。
その折り線は、従来の22.5度システムや円パッキングに比べて、はるかに予測しやすく管理しやすいものです。
純粋なボックスプリーツは紙面積の利用効率があまり高くないことで知られていますが、
ストレッチガジェット（神谷パターンやより一般化されたGOPSなど）の助けにより、
ボックスプリーツでも最適な円パッキングに近い効率を達成できるようになりました。
このアプリは、折り紙デザイナーがこれらのシステムでモデルの設計図を作成するのを支援するために作られました。

作者のブログで [Box Pleating Studio の物語](https://origami.abstreamace.com/2021/06/02/the-story-of-box-pleating-studio/)をご覧ください！
