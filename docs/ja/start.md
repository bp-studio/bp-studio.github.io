# はじめに

## 前提知識

Box Pleating Studio は折り紙の初心者向けではありません。
このアプリが生成するパターンを完全に理解し、それを折れるようになるには、
単軸ボックスプリーツ基本形の理論にかなり精通している必要があります。
Robert J. Lang の優れた著書の第13章と第14章に必要な知識がすべて記載されています
（参考文献 [[1]](/ja/notes#参考文献) を参照）。また、[リソース](/ja/resources)ページに掲載されている他のチュートリアルもご覧ください。

## アプリの入手方法

[https://bpstudio.abstreamace.com/](https://bpstudio.abstreamace.com/) にアクセスするだけで
BP Studio アプリを起動できます。ダウンロードは不要です！
最高のパフォーマンスを得るには、Chrome ブラウザの使用を強くお勧めします。
いずれにせよ、ブラウザを最新バージョンに更新してください。
BP Studio は多くの新しいWeb機能を使用しており、古いブラウザでは動作しません。

Chrome（iPhone以外のデバイス）、Edge（デスクトップ）、またはSafari（iPhone）でWebサイトにアクセスすると、
BP Studio をスタンドアロンアプリとしてデバイスにインストールし、オフラインで使用できることが案内されます。
指示に従ってインストールしてください。
これにより、特にモバイルデバイスで最高のユーザー体験が得られます。
