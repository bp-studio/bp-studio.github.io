## 謝辞

著者は、特に以下の方々・団体に感謝の意を表します（順不同）：

- [Robert J. Lang](https://langorigami.com/)：GOPS のアイデアに着想を与えた研究と、長年にわたる継続的なサポートと励ましに対して。
- [Lucas Tay Kiat Loong](https://cexplr.github.io/)：積極的に BP Studio のテストに協力してくださり、多くの重大かつ微妙なバグの発見に貢献し、数多くの貴重なフィードバックを提供してくださったことに対して。
- [Jason S. Ku](https://github.com/origamimagiro)：計算幾何学に関する洞察に対して。
- Mat Groves と [GoodBoyDigital](https://www.goodboydigital.com)：BP Studio を支えるグラフィックスエンジン [PixiJS](https://pixijs.com/) のメンテナンスに対して。
- [BrowserStack](https://www.browserstack.com/)：BP Studio をオープンソースプログラムに採用してくださったことに対して。これがなければ、BP Studio が互換性テストを行うことは費用的に不可能でした。

## 著者について

蔡牧村は Robert J. Lang の著書を読むことで折り紙のデザイン理論の研究を始めました。
二人はすぐに親しい友人となり、共同でピタゴラスストレッチの概念を GOPS へと一般化し、
それを 7OSME で発表しました。
蔡牧村は台湾在住で、フルスタック開発を生業としています。
特に C# と TypeScript を専門としています。
また、[Oriedita](https://oriedita.github.io/) プロジェクトの主要な貢献者でもあります。

ブログ [Tsai Origami](https://origami.abstreamace.com/) もぜひご覧ください！

[Avon Koh](https://www.youtube.com/@origameow) による著者への特別[インタビュー](https://youtu.be/-1usLhc3RNw)もご覧ください！
