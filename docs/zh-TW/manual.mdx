## 使用方式

當你建立新專案時，你會看到一個有兩條邊的基本樹狀結構
（這是應用程式中為了讓結構有意義而允許的最少邊數）。
首先，你應該決定你的模型應該有什麼樣的結構。
要編輯樹狀結構，點擊任意節點並使用選項面板中的指令來修改樹狀結構。
你可以新增節點、刪除葉節點、修改邊的長度，
以及執行一些進階操作，例如邊的分割和融合端點。
你也可以透過拖曳來重新排列紙張上節點的位置。
建議你為葉邊命名，以幫助辨識它們與角片的對應關係。

一旦你有了想要的結構，
前往佈局檢視並開始嘗試不同的角片位置組合。
BP Studio 會盡力為那些與對應矩形有重疊（但與對應圓形沒有重疊）的角片找到伸展圖案，
並嘗試自動決定所有角片和河的形狀。
關於本應用程式中產生伸展圖案的方法的簡要說明，請參見 [[2]](/zh-TW/notes#參考文獻) 和 [[3]](/zh-TW/notes#參考文獻)。

當你完成設計後，你可以儲存專案、儲存整個工作區，
或將設計匯出為 CP/[FOLD](https://github.com/edemaine/fold) 格式，以便在其他軟體中進一步編輯
（例如 [Oriedita](https://oriedita.github.io/)）。
請注意，匯出展開圖並不是為了產生可平折的 CP
（這目前超出了 BP Studio 的範圍），
而是為你提供一個起點，免去從零開始重新繪製的麻煩。

樞紐摺痕、脊線摺痕和軸平行摺痕的預設配色遵循 Lang 在其著作中的慣例，
但你也可以在喜好設定中自訂它們。
為了清晰起見，只會顯示伸展圖案的軸平行摺痕，
但你應該可以透過觀察脊線摺痕來輕鬆推斷其餘部分。

## 佈局最佳化

在工具選單中可以找到 v0.7 引入的「最佳化佈局」功能。
此工具使用類似 TreeMaker 的方法來為你的設計找到高效率的填充方案。
有兩種佈局方法：一種是使用你目前的佈局作為參考
（最佳化器會盡可能改善它），
另一種是讓最佳化器產生數個隨機佈局，並從中選用最有效率的一個。
在第一種方法中，還有一個選項「嘗試當前佈局的變化」，
它會嘗試稍微不同的角片排列方式，看看效率是否可以進一步提升。

請注意，最佳化器是純數值計算的，不會考慮填充的美觀性，
因此結果往往相當凌亂（但仍然有效）。
通常你需要手動重新排列結果以獲得整潔的佈局。

提示：計算的大多數階段都可以跳過。透過跳過某些步驟，
你也可以在完成樹狀結構建構後，使用此工具來簡單地將角片分散開來。
但這樣做可能會導致效率較低的佈局和無效的角片重疊。

## 目前的限制

BP Studio 並不總是能在每個有效的佈局中找到可行的伸展圖案。
目前作者已經實作了足夠的演算法，可以為任何兩個角片的有效佈局找到伸展圖案，
以及大多數三個角片的有效佈局。
作者將繼續為更複雜的伸展圖案實作更多演算法。

目前 BP Studio 尚未實作標高（elevation）、半整數單位結構或蜿蜒河的概念。
但你可以透過設定角片的寬度或高度來加寬或加高你的角片，
並利用它們在最終模型中創造標高變化。
你也可以使用整數結構來大致表示半整數結構所佔的空間，
然後在摺疊過程中將其轉換為後者。
蜿蜒河可以類似地用短角片來表示，
或者你可以直接假裝它在那裡，然後繼續摺你的作品就好。

其他功能如傾斜格線、最佳化條件、蜿蜒支援等也在規劃中。
