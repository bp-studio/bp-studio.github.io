## 簡介

箱形褶（Box pleating）可說是現代超複雜摺紙設計的標準框架，
因為其摺痕比傳統的 22.5 度系統或圓包裝更加可預測和易於管理。
雖然純箱形褶在紙張面積的利用效率上並不突出，
但在伸展結構（stretching gadgets）的幫助下（例如神谷圖案和更通用的 GOPS），
箱形褶現在已經可以達到接近最佳圓包裝的效率。
本應用程式旨在幫助摺紙設計師在這些系統中繪製其模型的藍圖。

歡迎閱讀作者部落格上的 [Box Pleating Studio 的故事](https://origami.abstreamace.com/2021/06/02/the-story-of-box-pleating-studio/)！
