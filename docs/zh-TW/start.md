# 入門指南

## 預備知識

Box Pleating Studio 並非為摺紙初學者設計。
為了充分理解它所產生的展開圖並能夠摺疊成品，
你必須對單軸箱形褶基本形的理論有相當程度的了解。
你可以在 Robert J. Lang 出色著作的第 13 和 14 章中找到所需的一切知識
（見參考文獻 [[1]](/zh-TW/notes#參考文獻)），或參考[資源](/zh-TW/resources)頁面中列出的其他教學。

## 如何取得應用程式

只需造訪 [https://bpstudio.abstreamace.com/](https://bpstudio.abstreamace.com/)
即可啟動 BP Studio 應用程式，無需下載！
為了獲得最佳效能，強烈建議使用 Chrome 瀏覽器來造訪網站。
無論如何，請將你的瀏覽器更新到最新版本，
因為 BP Studio 使用了許多新的網頁功能，無法在舊版瀏覽器上運行。

如果你使用 Chrome（iPhone 以外的裝置）、Edge（桌面版）或 Safari（iPhone）造訪網站，
它也會提示你可以將 BP Studio 安裝為獨立應用程式，並離線使用。
只需按照指示安裝即可。
這將提供最佳的使用者體驗，尤其是在行動裝置上。
