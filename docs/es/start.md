# Primeros pasos

## Requisitos previos

Box Pleating Studio no es para principiantes del origami.
Para comprender completamente el patrón que crea y poder plegarlo,
debes estar considerablemente familiarizado con la teoría de las bases de box pleating uniaxiales.
Encontrarás todo lo que necesitas saber en los capítulos 13 y 14 del notable libro de Robert J. Lang
(véase [[1]](/es/notes#bibliografía) en la sección de bibliografía), o consulta otros tutoriales en la sección de [Recursos](/es/resources).

## Cómo obtener la aplicación

Simplemente visita [https://bpstudio.abstreamace.com/](https://bpstudio.abstreamace.com/)
para iniciar la aplicación BP Studio, ¡sin necesidad de descargas!
Para el mejor rendimiento, se recomienda encarecidamente que uses Chrome para visitar el sitio.
En cualquier caso, actualiza tu navegador a la última versión,
ya que BP Studio utiliza muchas funciones web nuevas y no funcionará en navegadores obsoletos.

Si visitas el sitio web con Chrome (en dispositivos que no sean iPhone), Edge (en escritorios) o Safari (en un iPhone),
también te informará de que puedes instalar BP Studio en tu dispositivo como aplicación independiente y usarla sin conexión.
Sigue las instrucciones para instalarla.
Esto proporcionará la mejor experiencia de usuario, especialmente en dispositivos móviles.
